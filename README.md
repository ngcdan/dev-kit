# Dev-Kit Skills Central

Central repository for AI Agent skills and configurations, synced from local agent directories.

## Structure

- `gemini/` — Gemini config (`GEMINI.md`) and skills (`antigravity/skills/`)
- `codex/` — OpenClaw workspace folders
- `claude/` — Claude config files (`CLAUDE.md`, etc.)
- `scripts/` — Sync automation (see [scripts/README.md](scripts/README.md))

## Sync

```bash
./run.sh
```
